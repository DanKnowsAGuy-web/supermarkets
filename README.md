# Energy Plus: Supermarket Landing Page

Conversion-focused landing page for Energy Plus, an Independent Energy Solutions
Partner for supermarkets and grocery operators. Built with [Astro](https://astro.build).

**Live:** https://danknowsaguy-web.github.io/supermarkets/

## Develop

```bash
npm install
npm run dev      # local dev server
npm run build    # production build to dist/
npm run preview  # preview the production build
```

## Deploy

Pushing to `main` triggers a GitHub Actions workflow
(`.github/workflows/deploy.yml`) that builds the site and publishes it to
GitHub Pages. No manual step required — the live URL updates a minute or two
after each push.
